# 极简Agent平台启动说明

## 1. 后端启动步骤

### 填写阿里百炼API-Key
1. 打开配置文件：`src/main/resources/application.properties`
2. 找到阿里百炼相关配置：
   ```properties
   # 阿里百炼API配置
   spring.ai.openai.api-key=您的API-Key
   spring.ai.openai.base-url=https://dashscope.aliyuncs.com/compatible-mode/v1
   ```
3. 将`您的API-Key`替换为实际的阿里百炼API密钥

### 启动后端服务
```bash
# 进入项目目录
cd /home/user/vibecoding/workspace/spring-ai-bailiang-demo

# 使用Maven运行
mvn spring-boot:run

# 或打包后运行
mvn clean package
java -jar target/spring-ai-bailiang-demo-0.0.1-SNAPSHOT.jar
```

服务默认在`http://localhost:8080`启动

## 2. 前端页面使用

### 打开前端页面
直接在浏览器中打开HTML文件：
```
/home/user/vibecoding/workspace/spring-ai-bailiang-demo/frontend/index.html
```

### 连接后端配置
如果后端服务地址不是默认的`http://localhost:8080/api`，需要修改前端代码中的API地址：
1. 使用文本编辑器打开`frontend/index.html`
2. 找到`apiBaseUrl`变量定义：
   ```javascript
   const apiBaseUrl = 'http://localhost:8080/api';
   ```
3. 修改为实际的后端服务地址

## 3. 功能演示操作流程

### 知识库检索功能
1. 点击顶部导航栏的"知识库"标签
2. 在搜索框中输入关键词（如"公司"、"产品"等）
3. 点击搜索按钮或按Enter键
4. 查看搜索结果列表
5. 点击"查看详情"查看完整文档内容

### AI对话功能
1. 点击顶部导航栏的"AI对话"标签
2. 切换流式响应模式（左上角闪电图标）
3. 在输入框中输入问题
4. 点击发送按钮或按Enter键
5. 查看AI回复，观察流式响应效果

### 多Agent协作功能
1. 点击顶部导航栏的"多Agent协作"标签
2. 在输入框中输入复杂问题
3. 点击"开始协作"按钮
4. 观察问答Agent和总结Agent的协作过程
5. 查看最终的协作结果

## 4. 项目说明（小米百万亿Token计划演示材料）

### 项目概述
本项目是基于Spring Boot和Vue3构建的极简Agent平台，集成了阿里百炼AI服务，展示了RAG知识库、流式对话和多Agent协作等先进AI应用场景。

### 核心特性
- **RAG知识库**：基于内存存储的文档检索系统，支持关键词搜索和相关性排序
- **流式AI对话**：使用SSE技术实现实时流式响应，提升用户体验
- **多Agent协作**：模拟问答Agent和总结Agent协同工作，展示复杂任务分解能力

### 技术亮点
- 纯CDN前端架构，无需构建工具
- 内存虚拟数据存储，快速部署测试
- 统一响应封装和异常处理
- 优雅的UI设计和流畅的交互体验

### 应用场景
- AI应用快速原型开发
- 大语言模型能力演示
- 企业内部知识管理系统
- 智能客服系统集成

本项目可作为小米百万亿Token计划的演示案例，展示大模型在实际应用中的能力和价值。
